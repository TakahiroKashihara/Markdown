## 1. **登場人物**
- 中堅向け法人営業、新人研修、メザニンファイナンスなどの設計、組成、運営

## 2. **Q&A**
- **<font size = "4">メガバンクの法人営業との違い</font>**

相手にしている企業の規模が大きい。売り上げは50億以上。金融商品の取り扱い、口座開設、資産運用はしない。<font color = red>融資のみ</font>。ノルマは厳しくない。民業補完を目的とする。

- **<font size = "4">公共性の定義</font>**

自分で考える。赤字になってもやるということではない。

- **<font size = "4">法人営業</font>**

決まったルーチンはほぼない。2,30社を同時に担当。2,3月は融資審査に追われる。決算の内容を聞きに行く。

- **<font size = "4">営業で提供できるソリューション</font>**

通常の融資(長期。5-7年。)。全部融資だと財務諸表上よくないのでメザニンファイナンスを行う[^1]。海外の会社をM&A。
- **<font size = "4">DBJの役割</font>**

DBJは中立的。2社だと対立するが、DBJが第三軸として入る。

- **<font size = "4">地域の場合</font>**

小規模だが全国にある。情報を回収しやすい。

- **<font size = "4">出資について</font>**

普通の銀行に対しては、5%ルールがある。しかし、DBJにはその上限がない。
その出資にメザニンが使える。商品設計の方法は無数にあって楽しい。最近はメガバンクもやり始めた。
他にも、アセットファイナンス[^2]等を行う。


- **<font size = "4">働き方</font>**

社員で協力し合って、一から学ぶ。

- **<font size = "4">事業例</font>**

九州電力の350億円のプロジェクトファイナンス。どの関係者にどのリスクを寄せるかを契約書に書く。一年がかり。

- **<font size = "4">キャリア</font>**

千差万別。1回は営業をやる。30代後半になって自分の特徴が分かる時も多い。
- **<font size = "4">その他</font>**
- 新人は無知を自覚し、知ろうとすることが大事。
- DBJには正直な意見が求められている。適正なリスクリターン
- 同期は少ないが仲良くなるし、先輩とも仲良くなる。
[^1]:詳しくは、(https://www.dbj.jp/service/invest/mezzanine/) 参照。融資と株式の中間的な手法。リスクが高いが金利が高い。
[^2]:詳しくは、(https://www.dbj.jp/service/finance/asset_finance/)　参照。不動産の収益から返してもらう。
